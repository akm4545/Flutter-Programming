# **구글 애드몹**  
애드몹(Admob)은 구글에서 제공하는 광고 서비스이다. 간단한 가입만으로 누구든 광고를 게시하고 수익화를 할 수 있다. 플러터뿐만 
아니라 웹 및 다른 앱 개발 프레임워크에서도 자유롭게 사용할 수 있다. 플러터에서 사용할 수 있는 애드몹 광고 형태는 4가지다.
